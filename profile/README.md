# 10Seconds 🎙️

## How to run (react-native)
### Needs for running

```bash
ruby 2.7.6
node 18.16.0
platform : mac OS
```

### react-native-cli를 전역으로 설치
```bash
npm install -g react-native-cli
```

### node modules dependency 설치
```bash
npm install
```

### ios dependency 설치
```bash
cd ios
pod install
```

### 에뮬레이터 실행
```bash
npm run ios
```
